Project Overview

The Micro Satellite project represents a comprehensive approach to small satellite design, focusing on creating a cost-effective CubeSat platform that can serve multiple mission profiles. Our design emphasizes modularity, reliability, and innovative power management to extend mission duration and capability.

This project combines theoretical aerospace engineering principles with hands-on prototyping and testing, providing valuable experience in systems integration, orbital mechanics, and space-qualified component selection.

Technical Challenges

Power Management System

Developing an efficient power management system for the harsh space environment required careful consideration of solar panel placement, battery chemistry, and power distribution architecture.

Communication Subsystem

Implementing reliable communication with ground stations while managing power constraints and antenna deployment mechanisms in a compact form factor.

Thermal Management

Ensuring all components operate within temperature ranges despite extreme thermal cycling in low Earth orbit.

Key Features

  • 3U CubeSat form factor with modular payload bay
  • Advanced lithium-ion battery management system
  • Deployable solar panel arrays with MPPT controllers
  • S-band communication system with ground station compatibility
  • Attitude determination and control system (ADCS)
  • Onboard flight computer with redundant systems
  • Custom PCB design for space-qualified components

Development Process

Phase 1: Requirements Analysis

Conducted thorough mission requirements analysis, including orbit selection, payload requirements, and regulatory compliance for space missions.

Phase 2: System Design

Created detailed system architecture diagrams, component specifications, and interface control documents for all subsystems.

Phase 3: Prototyping & Testing

Built and tested individual subsystems, followed by integration testing and environmental qualification procedures.

// Attitude Control System - Main Loop
void attitude_control_loop() {
    // Read sensor data
    Vector3 gyro_data = read_gyroscope();
    Vector3 mag_data = read_magnetometer();
    
    // Estimate current attitude
    Quaternion current_attitude = estimate_attitude(gyro_data, mag_data);
    
    // Calculate control commands
    Vector3 control_torque = calculate_control_torque(
        current_attitude, 
        target_attitude
    );
    
    // Apply control to reaction wheels
    apply_wheel_torque(control_torque);
}
